# AD Preference Chatbot

An intelligent chatbot that answers questions about AD preferences, campaign performance, hype metrics, and company advertising trends using natural language.

## Features

- 🤖 Natural language understanding of AD-related queries
- 📊 Real-time PostgreSQL database querying
- 🔍 OpenAI-powered SQL generation
- 📈 Hype metrics and trend analysis
- 🏢 Company campaign performance tracking
- 📅 Date-based preference analysis

## Prerequisites

- Python 3.8+
- PostgreSQL database with AD/campaign data
- OpenAI API key
